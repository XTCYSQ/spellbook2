Частые вопросы

Что такое ИИ-ассистент? ⌃
ИИ-ассистент — это программа на базе нейросети, которая понимает естественный язык и может выполнять различные задачи: отвечать на вопросы, помогать с текстами, анализировать данные и многое другое.
Нужны ли технические знания для создания ассистента? ⌃
Нет, платформа SpellBook позволяет создавать ассистентов без программирования. Весь процесс происходит через простой визуальный интерфейс.
Могу ли я продавать своих ассистентов? ⌃
Да, вы можете публиковать ассистентов на маркетплейсе и получать доход от их использования другими пользователями.
Какие модели ИИ доступны? ⌃
Мы предоставляем доступ к последним версиям ChatGPT. Список постоянно обновляется.
Есть ли API для интеграции? ⌃
Нет, мы НЕ предоставляем REST API для интеграции ассистентов.
